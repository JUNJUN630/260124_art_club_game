# Vertical STG MVP (Pygame)

縦スクロールSTGの最小実装です。図形描画のみで動作します。

## 実行方法

```bash
python3 main.py
```

## 操作方法

- 移動: 矢印キー（8方向）
- ショット: Z
- リトライ: R（GAME OVER / CLEAR 時）

### デバッグ

- 当たり判定表示: C（ON/OFF）
- ベルドロップ率: [ で -5%、] で +5%
- ボススキップ: B（即ボス出現）

## 仕様メモ

- 30FPS 固定
- 論理解像度 320x288 / 整数倍スケール表示
- 状態: PLAYING / BOSS / GAMEOVER / CLEAR
- ライフ: 3。被弾で減少、0でゲームオーバー
- スコア: 敵撃破で加点のみ（倍率あり）

### 敵

- ジグザグ雑魚: 左右に振れながら下降
- 突撃雑魚: 1回だけ自機方向に突進
- 耐久雑魚: HP高め
- すべて等速弾を発射、体当たり判定あり

### ベル（アイテム）

- 敵撃破時に確率ドロップ
- ベルはショットで色が3色サイクル
- 自機接触で取得し効果適用
  - SPREAD: ショットが前方+斜めの3方向
  - RAPID: 連射間隔短縮
  - SCORE: スコア倍率アップ

### ボス

- 60秒の雑魚フェーズ後に出現
- HPあり、撃破でクリア
- 攻撃: 扇状ばらまき等速弾（7〜11発の角度分散）
